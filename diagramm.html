Diagrammid Projectis

kuidas luua ja kasutada diagramme MS Projectis

📹 Õppevideo: Täpsema selgituse leiad videost YouTube — Diagrammide loomine MS Projectis
1

Gantt-diagramm — peamine vaade

Gantt-diagramm on MS Projecti põhivaade. See näitab ülesandeid vasakul tabelis ja nende ajajoont paremal ribadena. Ava see: View → Gantt Chart.

📸 Lisa siia ekraanipilt: Gantt Chart vaade ülesannetega
2

Network Diagram (võrgustikudiagramm)

Võrgustikudiagramm näitab ülesannete vahelisi seoseid kastidena ja nooltena. Sobib projekti loogilise struktuuri mõistmiseks. Ava: View → Network Diagram.

📸 Lisa siia ekraanipilt: Network Diagram vaade
3

Resource Graph — ressursside koormus

Resource Graph näitab visuaalselt, kui palju tööd on igale ressursile määratud. Punane ala tähendab ülekoormust. Ava: View → Resource Graph.

📸 Lisa siia ekraanipilt: Resource Graph koos ylikoormuse näitajaga
4

Timeline — ajajoon

Timeline lisab lehe ülaossa kompaktse ajajoonesüsteemi. Saad sellele lohistada olulisemaid vahe-eesmärke. Luba: View → märgi linnuke Timeline.

📸 Lisa siia ekraanipilt: Timeline riba Gantt-diagrammi kohal
5

Diagrammi kohandamine

Gantt-ribasid saab värvida ülesande staatuse järgi: Format → Bar Styles. Saad ka lisada tekste ribade peale ja muuta kriitilise tee värvi (Format → Critical Tasks).

📸 Lisa siia ekraanipilt: Bar Styles dialoog või kriitilise tee vaade

Diagrammid aitavad projekti olekut kiiresti hinnata — vali igaks olukorraks sobiv vaade!
