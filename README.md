##  Sisu ülevaade

###  Kalender (`index.html`)
Juhend selgitab, kuidas luua kohandatud tööajakalender ja määrata see ülesannetele.

1. Avada **Change Working Time** (`Project → Change Working Time`)
2. Luua uus baasikalender (nt `TEST`)
3. Muuta tööaegu
4. Lisada erandid (**Exceptions**)
5. Vaadata tulemust Gantt-diagrammil

---

###  Arvutusväli (`valem.html`)
Juhend selgitab, kuidas lisada projekti kohandatud arvutusväli valemiga.

1. Avada **Custom Fields** (`Project → Custom Fields`)
2. Nimetada väli ümber (nt `Text1` → `arvutusklass`)
3. Sisestada valem (**Formula**)
4. Seadistada graafiline näidik (**Graphical Indicators**)
5. Vaadata tulemust tabelis

---

###  Diagrammid (`diagramm.html`)
Juhend tutvustab MS Projecti erinevaid diagramme ja aruandeid.

1. **Gantt Chart** (`View → Gantt Chart`)
2. **Reports** — Dashboards, Resources, Costs, In Progress
3. **Resource Cost Overview**
4. **Timeline** (`View → Timeline`)
5. **Report Design** — Chart, Table, Themes
