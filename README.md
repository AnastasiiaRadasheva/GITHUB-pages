##  Sisu ülevaade

###  Kalender (`index.html`)
Juhend selgitab 5 sammuga, kuidas:
1. Avada **Change Working Time** akna (`Project → Change Working Time`)
2. Luua uus baasikalender (nt `TEST`)
3. Vaadata ja muuta tööaegu
4. Lisada erand (**Exceptions**)
5. Vaadata tulemust Gantt-diagrammil

###  Arvutusväli (`valem.html`)
Juhend selgitab, kuidas:
1. Avada **Custom Fields** (`Project → Custom Fields`)
2. Nimetada väli ümber (nt `Text1` → `arvutusklass`)
3. Sisestada valem (**Formula**)
4. Seadistada graafiline näidik (**Graphical Indicators**)
5. Vaadata tulemust Gantt-tabelis

### Diagrammid (`diagramm.html`)
Juhend tutvustab 5 erineva vaatega:
1. **Gantt-diagramm** — põhivaade (`View → Gantt Chart`)
2. **Aruanded (Reports)** — Dashboards, Resources, Costs, In Progress
3. **Resource Cost Overview** — ressursside kulude diagramm ja tabel
4. **Timeline** — projekti ajajoon (`View → Timeline`)
5. **Report Design** — aruande kohandamine (Chart, Table, Themes)
